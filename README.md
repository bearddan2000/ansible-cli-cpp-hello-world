# ansible-cli-cpp-hello-world

## Description
The class c++ consol.

## Tech stack
- ansible
- gnu gcc 7.1.1
- g++

## Docker stack
- docker-cli
- ubuntu:latest

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
